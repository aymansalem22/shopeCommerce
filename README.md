# shope_Commerce

Spring Boot E-Commerce:

#Build a Real-life Shopping Webapp using Java Spring Boot, Thymeleaf, Bootstrap, jQuery and HTML :<br />

1. Develop a complete shopping website using Java, Spring Boot, Thymeleaf, Bootstrap, jQuery and MySQL database <br />
2. Code the Admin application for managing users, categories, brands, products, customers, orders, sales report...<br />
3. Code the Shopping application allowing customers to browse products and place orders<br />
4. Code shopping cart features and payment via PayPal and credit cards<br />
5. use Spring Data JPA with Hibernate for the data access layer<br />
6. use Spring Security for authentication and role-based authorization<br />
7. use Thymeleaf template engine for creating dynamic view pages<br />
8. use Bootstrap and jQuery for creating responsive, mobile-friendly web pages<br />
9. use Spring RESTful Webservices in conjunction with jQuery on the client side<br />
10. use PayPal Checkout API for payment integration<br />
11. use JUnit, Spring Test, AssertJ and Mockito for unit tests and integration tests<br />
12. deploy the application on Heroku cloud platform with file upload on Amazon S3<br />
13. Google Chart for drawing charts in sales report<br />
14. Spring OAuth for single sign on with Facebook and Google<br />
15. Spring Mail for sending emails for customer registration and place order confirmation<br />
